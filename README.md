# Operating Systems C Projects

Portfolio documentation for Operating Systems coursework completed through Marquette University.

These projects focused on lower-level programming concepts using C, UNIX/Linux development tools, process control, memory, file systems, and systems-level debugging.

## Course Context

This work was completed through Marquette Operating Systems coursework using private course repositories. The original class repositories may require course or organization access, so this public repo is used as a clean portfolio summary rather than a full public code dump.

## Focus Areas

- C programming
- UNIX/Linux development
- Process control
- Memory management concepts
- File systems
- Systems-level debugging
- Git and branch-based project workflow
- WSL/Linux command line development
